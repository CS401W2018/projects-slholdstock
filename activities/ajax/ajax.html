What is AJAX?

AJAX stands for Asynchronous Javascript And XML. It allows for interactive things to happen on your webpage without making the webpage reload. Data can also be sent and received easily.

Why do you put your JS Code in a separate file?

You put it in a separate file so that you can reuse it on other pages that you want to do the same thing. It also organizes everything easier so that you can focus on only one part of the page at a time.
