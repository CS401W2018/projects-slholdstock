Explain the difference between inline, embedded, and external CSS

Inline is done with the style attribute on an HTML tag.

First you put <p style=(property name):(value);> To get the red text the property name was color and the value was red.

Embedded CSS is when we go to the head section and add a <style> where we specify what we want it to change. In this case I did: p {font-family:arial, helvetica, sans-serif} to change the font of all the paragraphs to a different font.

External CSS is when we create a new file with the CSS specifications that we want. In order to bring it over to another page, we go to the <head> tag and type out <link rel="stylesheet" href="(name of file)">.

Explain why you should only use external CSS.

We design web sites, not web pages. We want them all to look the same. By creating a seperate file with a CSS stylesheet, we can easily link our other pages to the same stylesheet so that it all looks uniform and connected. When editing, we also only need to edit one thing instead of once on each page. Seperation of concerns comes into play.

What is a selector used for?

A selector is used to define which parts of the HTML should be styled. This would be like the colors we use or the fonts or boldness or lack thereof.
