What is the difference between an ID and a Class?

The main difference is that this will work, but an HTML validator will invalidate it. An ID can only be used once while a class can be repeated. A tag can also have more than one tag while the ID can only have one.

What is the difference between the SPAN and DIV tag

SPAN tags allow you to select a certain part of something like a paragraph to then change the appearance with CSS. A DIV tag allows you to add multiple tags. It also causes line breaks. It almost creates a box or block on the page.

What is the difference between a DIV tag and HEADER, MAIN, and FOOTER?

DIV used to be used like header, main, and footer are used now. When the guys got together and decided how HTML would be universally, they realized that people were constantly using DIV tags to do the same things, so now they have their own dedicated tags for their respective purpose.
