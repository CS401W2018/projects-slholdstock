What are special entity characters?

Special entity characters have different code so that they will show up on your page. They always start with "&" and end with ";"
For example:

> is written using the "code" & + gt + ; while < is written using the "code" & + lt + ;

© is written using the "code" & + "copy" + ;

What is the difference between OL and UL tags?

The OL tag will create an Ordered List while the UL tag creates an Unordered List.
