What is the difference between a block and an inline tag? Give examples of each one.

A block tag is something that makes an actual box on your webpage. You can add a border, a background color, or other things like that. An example would be the header tag, the main tag, or the border tag. An inline tag is something that cannot be given a height or a width. An example would be the anchor tags or image tags.

What is normal flow in HTML?

Normal flow is the default positioning of your HTML on your page. It says that a block tag demands the entire width of the page even if the contents don't.

What does float do? Why do you need to clear the float?

The float helps with positioning certain elements in your HTML. For example with a picture, it will put the image where we want it, then all the text "floats" around the rest of the unused space

What do absolute, fixed, and relative do?

Absolute is as if you took something (like an image) out of the page so that you could place it wherever you want. Relative is as if you take the image and place it relative to other things around it and it takes up the whole space. The fixed fixes an image to the edge of the browser page so it doesn't move as you scroll.

How can you make nice navigation buttons for mobile devices?

You can add background colors to the links and add a nav a:hover css property where you change or alternate the colors when the mouse is hovering over it.
