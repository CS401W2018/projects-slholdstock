How can you adjust the CSS for printing?

A trick you can use is to make a new css sheet with all of the same CSS or changing it slightly for however you want the printed page to be. You then link an additional CSS stylesheet but use the media tag to clarify that one is for the screen and one is for the print.

When would you want to adjust the CSS for printing?

You would do this if you know that a page that you have created will specifically be used to print things, for example tickets or something like that.
