Why is the viewport meta tag important?

it goes in the head tag that is automatically put in. It is what allows our media queries to work. If we don't have it, it will work on a desktop, but if we were to try it on a mobile device, it wouldn't work.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

"flex direction: row;" because it is the last thing that it was told to do so it is the last thing that will appear when it is put together. Whichever is there last is what shows up.

How do you change the CSS for a web page displayed on a phone?

You add media queries at the bottom of your css. Today we used the @media inquery so that screens that are smaller than a certain size will get the mobile version.
