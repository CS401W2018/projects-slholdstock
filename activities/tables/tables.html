Why don't we use the border property?

We don't use it anymore because it causes a validation error. It isn't used anymore in HTML because it is now something that CSS does that is better integrated into the webpage itself.

What is the easiest way to color every other row a different color?

The easiest way to do it is to use the "(class for your table) tbody tr:nth-child(even or odd)" then inside the brackets designate it a certain background color and adjust it to get what you want.

When should we use a table?

We can use it for a schedule like we have with ours. You could use it to show stats for players or the roster of a team. It is very good at lists when you are hoping to highlight certain things. If you have a lot of data, use a table.
