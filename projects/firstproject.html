Internet

The internet is a network of computers that connects with other computers around the world to share information.

Web

The web is a system of websites accessbile through the internet.

How are the Internet and Web different?

The internet is the network that connects everyone from wherever they are, and the web uses the internet to access information that is shared, like a website or emails.

ASCII and Unicode

Both ASCII and Unicode are character encoders meaning they have an assigned number value assigned to each character so that the computer can understand it because computers don't understand letters and words like we do. ASCII is not as universal as Unicode because Unicode supports various languages.

What is UTF-8?

UTF-8 or Unicode Transformation Format - 8 bits, is an encoding system for Unicode. What it does is transform a character from Unicode to a binary string and vice versa. It is useful because it saves memory.

URL

URL, or Uniform Resource Locator, is an address that one uses to gain access to certain resources on the internet.

Domain Name

The domain name is the part of your URL that is like the main part that the computer usnderstands as an IP address to show you what you are searching for. For example: www.byuh.edu

HTTP

HTTP stands for Hypertext Transfer Protocol which is used by the web to transmit information for certain things like pictures and videos.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Seccure and is essentially the same as HTTP, but with more security. The data is encrypted so that it is harder to tamper with.

IP Address

An IP Address or Internet Protocol Address is a set of numbers that uses 32 or 128 bits that is assigned to a device on the internet. The device then becomes recognizable by the internet so that it can communicate with other devices.

DNS

DNS stands for Domain Name System is what translates a domain name into an IP Address that the computer can use to connect with others. If you type a domain name like www.byuh.edu, DNS will turn it into the IP Address that the computer understands.

W3C Recommendations

W3C Recommendations are a set of web standards that have been designed to optimize accessiblity and ensure quality.

XML, XHTML, HTML, HTML5

XML is eXtensible Markup Language which is a language that makes it so that both humans and computers can understand something using tags that the user defines. XHTML is eXtensible Hypertext Markup Language which is a type of HTML and XML crossbreed. It exists to have more reliable web documents that work well with other formats. HTML is Hypertext Markup Language is a standard for documents and pages designed for the web. HTML5 is the latest version of HTML that has been improved over the years.
